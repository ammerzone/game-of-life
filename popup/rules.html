Spielregeln

Eine lebendige Zelle stirbt, wenn...

  • ... sie weniger als zwei lebendige Nachbarzellen hat (Einsamkeit).
  • ... sie mehr als drei lebendige Nachbarzellen hat (Überbevölkerung).

Eine Zelle überlebt, wenn sie zwei oder drei lebendige Nachbarn hat.

Eine Zelle wird wiederbelebt, wenn sie genau drei lebendige Nachbarn hat.
